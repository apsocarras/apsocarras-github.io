About Me

I am a 2023 Data Engineering Fellow at the Data Innovation Lab at Tech Impact, a non-profit data science consultancy assisting other non-profits and government partners in leveraging data for social good. Prior to a shift in career goals, I was a data analyst working in public health and preparing for Ph.D. programs in biostatistics. I eventually realized that working in software was more exciting to me than academic research.

I am primarily seeking Data Engineering, Cloud Engineering, and Python/SQL Developer roles where I can continue to progress as a developer working with data. My experience in public advocacy and consulting additionally makes me well-suited for client-facing positions where I can fully leverage my broad skill sets in communication and project management.

Benefits Cliffs

(WIP) An early-stage research collaboration with the Atlanta Federal Reserve Bank to create a dashboard visualizing benefits cliffs across the US.

I wrote a Python class that interacts with and expands on an R script the AFRB produced to estimate benefits and expenses for a given American family.

I'm using this and other tools I'm developing to build out a custom dashboard in Dash that integrates income projections for jobs in the BLS database and a skills assessment for job seekers.

The current state of the project is housed in a private GitHub repo, but we'll publish more work when the project is further along.

(GitHub currently private)

Survey App

I wrote and deployed a serverless Flask app in Azure to connect a Workforce Development Board survey in SurveyMonkey to an external API for processing and feedback.

I also conducted extensive data analysis on the survey responses (WIP).
